# AgileStock

CSC480A - AgileStock inventory management application

## Development Environment

1. Clone the repro to a local folder
   ```
   git clone https://github.com/robhowe-A/AgileStock.git
   ```
2. Start the web server
   ```
   py -m runserver
   ```

### Dependencies

- This project runs with Python, be sure have Python 3.11 installed on the server. It requires Flask, so be sure to install that, too:

```
py -m pip install Flask
```
